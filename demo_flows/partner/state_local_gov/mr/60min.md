Title: Meraki MR Demo for State Local Gov Sector

Intro (5 mins):
- Welcome partners to the demo
- Briefly introduce Meraki and its cloud-managed networking solutions
- Set the stage for the demo focusing on Meraki MR access points

Pain Points & Positioning (10 mins):
- Discuss common pain points in State Local Gov sector such as limited IT resources, complex network management, and security concerns
- Position Meraki as a solution that offers easy deployment, centralized management, and robust security features
- Emphasize the cost savings and efficiency benefits of cloud-managed networking

Use Case 1: Secure Guest Wi-Fi (10 mins):
- Demonstrate how partners can **click 'Configure > Wireless > SSIDs'** to set up a separate guest network
- Showcase the built-in splash page customization and guest access controls
- Explain how this feature can help State Local Gov entities provide secure and seamless Wi-Fi access to visitors

Use Case 2: Location Analytics (10 mins):
- Show partners how to **click 'Monitor > Location Analytics'** to track foot traffic and dwell times in different areas
- Discuss how this data can be used by State Local Gov agencies to optimize space utilization and improve visitor experiences
- Highlight the privacy-conscious nature of Meraki's analytics tools

Use Case 3: Application Visibility (10 mins):
- Guide partners to **click 'Monitor > Application Visibility'** to view real-time application usage on the network
- Explain how this feature can help State Local Gov organizations prioritize critical applications and identify potential security threats
- Discuss the insights gained from granular application data

Use Case 4: Remote Troubleshooting (10 mins):
- Illustrate how partners can **click 'Monitor > Clients'** to troubleshoot connectivity issues for specific devices
- Showcase the remote packet capture and live tools available for diagnosing network problems
- Stress the time-saving benefits of remote troubleshooting for State Local Gov IT teams

Smart Questions for Discussion:
1. How have you traditionally managed your network infrastructure, and what challenges have you faced?
2. What specific security requirements do you have for your network, and how do you currently address them?
3. How do you see cloud-managed networking solutions like Meraki fitting into your long-term IT strategy?

Conclusion (5 mins):
- Recap key features and benefits of Meraki MR for State Local Gov sector
- Invite partners to reach out for further information or assistance
- Thank attendees for their time and participation in the demo

End of Script.
